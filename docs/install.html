Linux

1. Download the .AppImage file

2. Make it executable:

chmod +x CapyDeploy_*.AppImage

3. Run it — it will prompt to install automatically

Installs to ~/.local/bin/ with desktop entry and icon.

CLI: --install, --uninstall, --help

Windows

1. Download the .zip file

2. Extract the ZIP

3. Run capydeploy-hub.exe or capydeploy-agent.exe

WebView2 is pre-installed on Windows 10/11.

For contributors

Building from Source

For contributors and developers who want to build from source code.

1

Install Prerequisites

# Install Go 1.24+ and Wails CLI
go install github.com/wailsapp/wails/v2/cmd/wails@latest
wails doctor
2

Clone and Build

git clone https://github.com/lobinuxsoft/capydeploy
cd capydeploy

# Build Hub (your PC)
cd apps/hub && ./build.sh

# Build Agent (handheld)
cd apps/agents/desktop && ./build.sh
3

Run and Pair

# On handheld
./capydeploy-agent
# Shows 6-digit pairing code

# On PC
./capydeploy-hub
# Enter the pairing code â†’ Done!
