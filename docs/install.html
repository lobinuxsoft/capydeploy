Installation

Get CapyDeploy running in a few steps.

1

Install Prerequisites

# Install Go 1.24+ and Wails CLI
go install github.com/wailsapp/wails/v2/cmd/wails@latest
wails doctor
2

Clone and Build

git clone https://github.com/lobinuxsoft/capydeploy
cd capydeploy

# Build Hub (your PC)
cd apps/hub && ./build.sh

# Build Agent (handheld)
cd apps/agents/desktop && ./build.sh
3

Run and Pair

# On handheld
./capydeploy-agent
# Shows 6-digit pairing code

# On PC
./capydeploy-hub
# Enter the pairing code → Done!

AppImage (Linux)

For easy distribution on Linux without dependencies.

Build AppImage

# Build Hub (includes AppImage)
cd apps/hub && ./build.sh

# Build Agent (includes AppImage)
cd apps/agents/desktop && ./build.sh

Output: dist/appimage/CapyDeploy_Hub.AppImage

Auto-Installation

Double-click the AppImage and it will prompt to install:

  • • Moves to ~/.local/bin/
  • • Creates desktop entry in ~/.local/share/applications/
  • • Copies icon to ~/.local/share/icons/
# Command line options
./CapyDeploy_Agent.AppImage --install    # Install manually
./CapyDeploy_Agent.AppImage --uninstall  # Remove
./CapyDeploy_Agent.AppImage --help       # Show options

Decky Plugin (Gaming Mode)

Alternative Agent that runs inside Decky Loader — no Steam restart needed.

Build & Install

# Build (requires Node.js + Python 3)
cd apps/agents/decky && ./build.sh
# Output: dist/decky/CapyDeploy-vX.Y.Z.zip

# Install on handheld:
# Decky Settings > Install Plugin from ZIP
Agent vs Decky Plugin
ShortcutsInstant (SteamClient API)
ArtworkInstant (no file copy)
UIQuick Access Menu panel
Steam restartNot needed

Platform Dependencies (for building from source)

Fedora / Bazzite

rpm-ostree install webkit2gtk4.0-devel gtk3-devel

Ubuntu / Debian

apt install libwebkit2gtk-4.0-dev libgtk-3-dev

Arch Linux

pacman -S webkit2gtk gtk3

Windows

WebView2 pre-installed on Win10/11
