#title
CapyDeploy

#description
Deploy games to your handheld devices with the chill energy of a capybara. :br
**WebSocket-powered.** Real-time progress. Auto Steam shortcuts. Zero stress.

#extra
  ::list
  - **Auto-discovery** via mDNS - no manual IP config
  - **WebSocket communication** - real-time, bidirectional
  - **Chunked binary uploads** with resume support
  - **Steam integration** - shortcuts & artwork
  - **Secure pairing** with 6-digit codes + persistent tokens
  - **Cross-platform** - Windows, Linux, Steam Deck
  ::

#support
  ::terminal
  ---
  content:
  - ./capyhub    # Run the Hub on your PC
  - ./capyagent  # Run the Agent on your handheld
  - # WebSocket connects automatically!
  ---
  ::
::

#title
Why CapyDeploy?

#root
:ellipsis{left=0px width=40rem top=10rem blur=140px}

#default
  ::card{icon=ph:broadcast-duotone}
  #title
  Auto-Discovery
  #description
  Agents broadcast via mDNS (`_capydeploy._tcp`). Hub finds them instantly. No IP addresses needed.
  ::

  ::card{icon=ph:plugs-connected-duotone}
  #title
  WebSocket Protocol
  #description
  Persistent bidirectional connection. Real-time progress events. Binary chunk transfers. No polling.
  ::

  ::card{icon=ph:game-controller-duotone}
  #title
  Steam Integration
  #description
  Creates shortcuts automatically. Downloads and applies artwork from SteamGridDB. Restarts Steam when needed.
  ::

  ::card{icon=ph:key-duotone}
  #title
  Secure Pairing
  #description
  6-digit code on first connect. Token stored locally for future sessions. No passwords to remember.
  ::

  ::card{icon=ph:lightning-duotone}
  #title
  Binary Uploads
  #description
  Games sent as 1MB binary chunks over WebSocket. Resume on disconnect. Progress events in real-time.
  ::

  ::card{icon=ph:devices-duotone}
  #title
  Cross-Platform
  #description
  Windows Hub → Linux Agent. Linux Hub → Windows Agent. Steam Deck, Bazzite, any Steam device.
  ::
::

How It Works

#root

#default
  ::card{icon=ph:number-circle-one-duotone}
  #title
  Agent Broadcasts
  #description

  ::

  ::card{icon=ph:number-circle-two-duotone}
  #title
  Hub Discovers & Connects
  #description
  Hub finds Agent, opens WebSocket connection (`ws://agent:9999/ws`).
  ::

  ::card{icon=ph:number-circle-three-duotone}
  #title
  Pair & Deploy
  #description
