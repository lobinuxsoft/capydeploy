# REST API

::alert{type="warning"}
**Deprecated.** These HTTP endpoints exist for backwards compatibility. Use the [WebSocket API](/api/websocket) instead for all new integrations.
::

The Agent exposes HTTP endpoints on port **9999**.

**Base URL:** `http://<agent-ip>:9999`

## Public Endpoints

Always accessible, even when connections are blocked.

### GET /health

Health check.

```json [Response]
{ "status": "ok" }
```

### GET /info

Agent information including capabilities.

```json [Response]
{
  "id": "agent-abc123",
  "name": "Steam Deck",
  "platform": "linux",
  "version": "0.1.0",
  "acceptConnections": true,
  "supportedImageFormats": ["png", "jpg", "webp", "gif"],
  "capabilities": ["file_upload", "steam_shortcuts", "..."]
}
```

### GET /config

Agent configuration.

```json [Response]
{ "installPath": "/home/deck/Games" }
```

## Protected Endpoints

::alert{type="danger"}
**Deprecated.** These endpoints require "Accept Connections" enabled. Prefer WebSocket equivalents.
::

| Endpoint | WebSocket Equivalent |
|----------|---------------------|
| `GET /steam/users` | `get_steam_users` message |
| `GET /shortcuts/{userID}` | `list_shortcuts` message |
| `POST /shortcuts/{userID}` | `create_shortcut` message |
| `DELETE /shortcuts/{userID}/{appID}` | `delete_shortcut` message |
| `POST /shortcuts/{userID}/{appID}/artwork` | `apply_artwork` message |
| `POST /steam/restart` | `restart_steam` message |

## Upload Endpoints

::alert{type="danger"}
**Deprecated.** File uploads should use WebSocket binary messages for better performance and real-time progress.
::

| Endpoint | WebSocket Equivalent |
|----------|---------------------|
| `POST /uploads` | `init_upload` message |
| `POST /uploads/{id}/chunks` | Binary WebSocket message |
| `POST /uploads/{id}/complete` | `complete_upload` message |
| `DELETE /uploads/{id}` | `cancel_upload` message |

## Error Responses

```json
{ "error": "error message here" }
```

| Code | Meaning |
|------|---------|
| 200 | Success |
| 201 | Created |
| 400 | Bad Request |
| 404 | Not Found |
| 500 | Internal Error |
| 503 | Connections blocked |
